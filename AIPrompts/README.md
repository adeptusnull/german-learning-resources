# AI Prompts

This folder contains AI prompt templates and collections for German language learning.

## What to Add Here

Add AI prompts designed to help with German language practice and learning:

- **Conversation prompts**: Templates for practicing German conversations with AI assistants (ChatGPT, Claude, etc.)
- **Grammar practice prompts**: Prompts for explaining and practicing German grammar concepts
- **Translation exercises**: Prompts for translation practice and corrections
- **Writing assistance**: Prompts for German essay writing, letter composition, and text improvement
- **Vocabulary building**: Prompts for learning new words, creating example sentences, and mnemonics
- **Pronunciation help**: Prompts for phonetic explanations and pronunciation guidance
- **Cultural context**: Prompts for understanding German culture, idioms, and expressions
- **Language level assessment**: Prompts to evaluate and improve language proficiency
- **Custom learning plans**: Prompts to generate personalized study schedules and materials
- **Error correction**: Prompts for identifying and explaining German language mistakes

## Preferred File Formats

- Markdown (.md) - preferred for easy viewing and editing
- TXT - for simple prompt collections
- JSON - for structured prompt templates that can be imported into AI tools

## Organization Suggestions

Consider organizing prompts by:
- Purpose (conversation, grammar, writing, etc.)
- Proficiency level (A1, A2, B1, B2, C1, C2)
- AI platform (ChatGPT, Claude, Gemini, etc.)
- Topic or theme

## Tips for Contributors

- Write clear, specific prompts that produce useful learning outputs
- Include example outputs or use cases where helpful
- Specify any particular AI model or version the prompt was optimized for
- Add context about the learning goal of each prompt
- Test prompts before submitting to ensure they work as intended
- Include variables or placeholders where users can customize prompts (e.g., [YOUR_WORD], [TOPIC])
- Provide instructions on how to use the prompts effectively

## Ethical Considerations

- Remember that AI is a learning aid, not a replacement for study
- Use AI prompts to enhance understanding, not to bypass learning
- Always verify AI-generated content with reliable sources
- Practice active learning by engaging with the AI's responses
