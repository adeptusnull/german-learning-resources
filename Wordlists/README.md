# Wordlists

This folder contains organized vocabulary lists for German language learning.

## What to Add Here

Add curated word collections:

- **Frequency lists**: Most common German words (top 100, 500, 1000, etc.)
- **Thematic lists**: Words grouped by topic (food, family, work, travel, emotions, etc.)
- **Verb lists**: Common verbs with conjugation information and separable prefixes
- **Noun lists**: With articles (der/die/das) and plural forms
- **Adjective lists**: Common descriptive words with comparative forms
- **Preposition lists**: With case requirements and common usage
- **Academic vocabulary**: For university-level German
- **Business German**: Professional and workplace terminology
- **Cognates**: German-English words with shared roots

## Preferred File Formats

- Markdown (.md) - preferred for tables and easy viewing
- CSV - for data that can be imported into apps
- Excel/Google Sheets - for complex vocabulary data
- JSON - for programmatic access

## File Organization

Subfolders are organized by word type:
- `Verbs/` - Verb lists with conjugation info
- `Nouns/` - Nouns with articles and plural forms
- `Adjectives/` - Descriptive words and their comparatives

## Tips for Contributors

- Use tables for structured data (word, translation, gender, example, etc.)
- For nouns, ALWAYS include the article (der/die/das) and plural form
- For verbs, indicate if separable and provide key conjugations
- Include example sentences for context
- Note regional variations (Swiss German, Austrian German, etc.) if applicable
- Organize alphabetically or by frequency
- Tag with CEFR levels when possible (A1, A2, B1, B2, C1, C2)
