# Resource Links

This folder contains curated collections of external resources for German language learning.

## What to Add Here

Add organized lists of helpful online resources:

- **Websites**: German learning platforms, news sites in simplified German, online dictionaries
- **YouTube channels**: Language learning channels, German content creators, educational videos
- **Podcasts**: German language podcasts for different proficiency levels
- **Apps**: Mobile apps for vocabulary, grammar practice, language exchange
- **Online courses**: MOOCs, university courses, free learning platforms
- **Communities**: Forums, Discord servers, Reddit communities, language exchange platforms
- **Tools**: Translation tools, conjugation checkers, grammar analyzers

## Preferred File Formats

- Markdown (.md) - preferred for easy viewing and editing

## Tips for Contributors

- Organize links by category or proficiency level
- Include brief descriptions for each resource
- Indicate if resources are free or paid
- Note the language of instruction (English, German, multilingual)
- Check links periodically and remove dead links
- Add your personal experience/review if you've used the resource
