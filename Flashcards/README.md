# Flashcards

This folder contains flashcard decks for German vocabulary and grammar practice.

## What to Add Here

Add flashcard collections for spaced repetition learning:

- **Vocabulary decks**: Themed word lists (food, travel, business, daily life, etc.)
- **Verb conjugations**: Practice sets for common verbs in different tenses
- **Noun genders**: Articles and noun gender practice
- **Phrases and idioms**: Common German expressions and their meanings
- **Grammar concepts**: Cases, prepositions, adjective endings
- **Sentence examples**: Context-based learning with full sentences

## Preferred File Formats

- **Anki** (.apkg) - Anki deck files
- **CSV** - Universal format that can be imported into various flashcard apps
- **TSV** - Tab-separated values for easy import
- **Quizlet** - Links to Quizlet sets or exported data
- **Markdown** - Simple text format with Q&A pairs

## File Organization

Consider organizing by:
- Proficiency level (A1, A2, B1, B2, C1, C2)
- Theme or topic
- Frequency (most common words first)

## Tips for Contributors

- Include both German to English and English to German cards
- Add pronunciation hints using IPA if possible
- Include example sentences for context
- Tag cards with proficiency levels
- Provide deck descriptions in the filename or accompanying README
- For noun decks, always include the article (der/die/das)
