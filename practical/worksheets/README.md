## üéØ How to Use These Worksheets

### For Self-Study:

**Before Watching:**
1. Read the context and summary
2. Study the key vocabulary list
3. Review cultural notes
4. Prepare note-taking materials

**During Watching:**
1. Use the active viewing templates
2. Pause to fill in observations
3. Note unfamiliar vocabulary
4. Complete comprehension checkpoints

**After Watching:**
1. Answer all episode-specific questions
2. Work through language analysis sections
3. Complete post-viewing activities
4. Check your answers with the answer keys
5. Do the writing and speaking exercises

**Review:**
1. Watch the episode again without pausing
2. Review your vocabulary notes
3. Practice pronunciation
4. Discuss with others or write reflections

### For Classroom Use:

**Homework Assignment (Before Class):**
- Assign pre-viewing section
- Have students study vocabulary
- Optional: Students watch episode at home with notes

**In-Class Activities:**
- Watch episode together (or key scenes)
- Discuss comprehension questions as a group
- Practice role-play scenarios in pairs
- Review grammar points collectively

**Homework Assignment (After Class):**
- Complete remaining questions
- Write assigned essays
- Prepare discussion points for next class

### For Study Groups:

**Group Session Structure:**
1. **Review vocabulary together** (15 min)
2. **Watch episode as a group** (45 min)
   - Pause for particularly important scenes
   - Discuss confusing moments
3. **Break and discuss** (15 min)
4. **Answer questions together** (30 min)
   - Share different perspectives
   - Help each other understand
5. **Role-play exercises** (20 min)
   - Practice dialogue scenes
   - Rotate roles
6. **Assign individual homework** (5 min)
   - Writing tasks
   - Personal vocabulary review
