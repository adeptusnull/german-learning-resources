## üóÇÔ∏è Repository Structure

The repository is organized into the following main categories:

Visual learning materials including grammar charts, vocabulary posters, and illustrated guides. Perfect for visual learners who benefit from diagrams and graphical representations of language concepts.

**Examples:** Cases overview diagrams, verb conjugation charts, vocabulary mind maps

### [`/Cheatsheets`](./Cheatsheets)
Quick-reference materials for grammar, pronunciation, common phrases, and essential language rules. Ideal for quick lookups and review sessions.

**Examples:** Grammar summaries, pronunciation guides, common phrase collections

### [`/ResourceLinks`](./ResourceLinks)
Curated collections of external resources including websites, YouTube channels, podcasts, apps, and online courses. A comprehensive directory of German learning tools available across the web.

**Examples:** Learning platform recommendations, podcast lists, community forums

### [`/Flashcards`](./Flashcards)
Flashcard decks in various formats (Anki, CSV, etc.) for vocabulary acquisition and grammar practice through spaced repetition learning.

**Examples:** Themed vocabulary decks, verb conjugation practice sets, common phrases

### [`/LearningGuides`](./LearningGuides)
Comprehensive study plans, detailed grammar explanations, and structured learning curricula. These guides provide in-depth coverage of German language topics.

**Examples:** 30-day study plans, grammar deep-dives, exam preparation guides

### [`/Wordlists`](./Wordlists)
Organized vocabulary collections including frequency lists, thematic word groups, and comprehensive verb/noun/adjective compilations with grammatical information.

**Examples:** Top 500 most common words, themed vocabulary (travel, food, etc.), verb lists with conjugations

### [`/AIPrompts`](./AIPrompts)
AI prompt templates and collections for German language learning with AI assistants like ChatGPT, Claude, and others. Includes prompts for conversation practice, grammar explanations, writing assistance, and more.

**Examples:** Conversation practice prompts, grammar explanation templates, vocabulary building exercises

## ü§ù Contributing

We welcome contributions from German learners and teachers alike! Here's how you can help improve this repository:

### What to Contribute

- **Infographics**: PDF or high-resolution image files explaining German grammar or vocabulary concepts
- **Cheatsheets**: Concise reference materials in Markdown, PDF, or DOCX format
- **Resource Links**: Well-organized lists of helpful websites, apps, videos, and courses
- **Flashcards**: Anki decks, CSV files, or other flashcard formats for vocabulary practice
- **Learning Guides**: Study plans, grammar guides, or comprehensive learning materials
- **Wordlists**: Organized vocabulary lists with translations and grammatical information
- **AI Prompts**: Prompt templates for using AI assistants to practice and learn German

### Guidelines for Contributors

1. **Quality over Quantity**: Focus on accurate, well-organized, and genuinely helpful materials
2. **Clear Organization**: Use descriptive filenames and place resources in the appropriate folder
3. **Include Context**: Add descriptions or README updates to explain what you're adding
4. **Proper Attribution**: Credit original sources and respect copyright/licensing
5. **File Formats**: 
   - Prefer Markdown (.md) for text-based content
   - Use PDF for printable/formatted documents
   - Ensure images are high quality and readable
6. **Language Levels**: When applicable, indicate CEFR proficiency levels (A1, A2, B1, B2, C1, C2)
7. **Grammar Details**: For nouns, always include articles (der/die/das) and plural forms

### How to Contribute

1. Fork this repository
2. Add your resources to the appropriate folder
3. Update relevant README files if needed
4. Create a pull request with a clear description of what you're adding
5. Ensure your contributions follow the guidelines above

### Need Help?

- Check the README.md file in each folder for specific guidance on what to add
- Look at existing resources as examples
- Open an issue if you have questions or suggestions
