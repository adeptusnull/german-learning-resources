# Learning Guides

This folder contains comprehensive study guides and learning plans for German language acquisition.

## What to Add Here

Add structured learning materials and study plans:

- **Study plans**: 30-day challenges, semester-long curricula, self-study roadmaps
- **Grammar guides**: In-depth explanations of German grammar topics
- **Pronunciation guides**: Detailed phonetics and accent training materials
- **Writing guides**: Letter writing, essay structure, formal vs. informal writing
- **Speaking guides**: Conversation strategies, pronunciation practice, common mistakes
- **Cultural guides**: German customs, etiquette, cultural context for language learning
- **Exam preparation**: TestDaF, Goethe-Zertifikat, telc preparation materials
- **Progress tracking**: Worksheets and templates for monitoring learning progress

## Preferred File Formats

- Markdown (.md) - preferred for easy viewing and version control
- PDF - for formatted study materials
- DOCX - for editable guides

## Tips for Contributors

- Structure guides with clear headings and progression
- Include practical exercises or practice suggestions
- Reference CEFR levels (A1-C2) where applicable
- Add learning objectives at the beginning of each guide
- Include recommended time commitments
- Link to external resources when appropriate
- Provide answer keys for exercises when possible
