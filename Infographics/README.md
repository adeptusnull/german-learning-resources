# Infographics

This folder contains visual learning aids and infographics for German language learning.

## What to Add Here

Add visual resources that help explain German language concepts, such as:

- **Grammar infographics**: Visual explanations of German grammar rules (cases, verb conjugations, sentence structure, etc.)
- **Vocabulary posters**: Themed vocabulary collections with images (e.g., food, animals, colors, body parts)
- **Timeline charts**: Historical language development, tense usage timelines
- **Comparison charts**: German vs. English grammar comparisons
- **Mind maps**: Visual connections between related concepts

## Preferred File Formats

- PDF (vector-based preferred for quality)
- PNG or JPG (high resolution, at least 1920x1080)
- SVG (for scalable graphics)

## Subfolders

- `Grammar/` - Infographics related to German grammar concepts
- `Vocabulary/` - Visual vocabulary aids and themed collections

## Tips for Contributors

- Ensure images are clear and readable
- Include sources or credits where applicable
- Use descriptive filenames (e.g., `german-cases-overview.pdf`)
- Add a brief description in commit messages
