# Cheatsheets

This folder contains quick-reference cheatsheets for German language learning.

## What to Add Here

Add concise reference materials for quick lookup:

- **Grammar cheatsheets**: Quick references for cases, articles, verb conjugations, prepositions
- **Pronunciation guides**: IPA charts, pronunciation rules, common sound patterns
- **Common phrases**: Useful expressions for different situations (greetings, shopping, travel)
- **Number and time**: Cardinal/ordinal numbers, time expressions, dates
- **Sentence structure**: Word order rules, question formation, subordinate clauses

## Preferred File Formats

- Markdown (.md) - preferred for easy viewing on GitHub
- PDF - for formatted printable versions
- DOCX - for editable documents

## Tips for Contributors

- Keep content concise and scannable
- Use tables and bullet points for clarity
- Include examples wherever possible
- Organize by proficiency level if applicable (A1, A2, B1, B2, C1, C2)
- Use descriptive filenames (e.g., `german-verb-conjugation-guide.md`)
